Так же, как и HTML, CSS прошел через множество итераций. Также, как и спецификация HTML, спецификация CSS поддерживается W3C. CSS был разработан для того, чтобы отделить дизайн от контента. Контент помещается в HTML, а CSS придает этому контенту привлекательный вид.
CSS (Cascading Style Sheets, Каскадные таблицы стилей) – это способ изменять внешний вид HTML-страницы.
HTML5 и CSS3
На самом деле, CSS необязательно применять только к HTML, но это, безусловно, его основное применение.
